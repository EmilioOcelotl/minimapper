Instrucciones

MiniMapper funciona mejor en Chrome o Firefox.

Hydra: Escribe código en el editor y haz clic en RUN para ver los cambios

Fuente: Cada quad puede usar Hydra, Video o Imagen como textura — elige con el selector y carga el archivo con el botón "Cargar"

Quads: Clic en + Quad, luego dibuja el área en el canvas con clic y arrastre. ESC cancela

Formas libres: Clic en + Libre, luego clic para colocar cada vértice. Cierra con doble clic en cualquier parte, o clic en el primer vértice (se vuelve verde). Mínimo 3 vértices. ESC cancela

Mapping: Arrastra los puntos de control para deformar la superficie. Las esquinas (rojo) están sobre la superficie; los puntos intermedios (amarillo) curvan los bordes

Atajos: CTRL + SHIFT + H para mostrar/ocultar la UI · ESC para silenciar o cancelar

Auto-guardado: El código y los vértices se guardan automáticamente (las fuentes de video/imagen se deben recargar al abrir)

Sesión: "↓ Sesión" descarga un archivo JSON con el estado completo (imágenes incluidas). "↑ Sesión" lo restaura — los videos y la cámara se deben recargar manualmente
